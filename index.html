Git basics

adding git to your existing project

  • git init
  • git add .
  • git commit -m “initial commit”

change some code

  • git add .
  • git commit -m “changed cuz”

Share your code to github

  • create github account
  • go to github and create new repository with the name of your project
  • connect to your github account from c9 dashboard
  • do NOT initialize with readme (since we have existing project already)
  • assuming everythign is committed...follow instructions on the screen starting with 
    "remote add"
    (since you did the other stuff already)
  • when you do 
    git push
    it'll ask for your username and password (note, your password won't show anything as you type, just press enter).

Collaborating on a project

  • go to https://github.com/sgelbart/html-playground
  • click “fork”
  • c9 -> repositories -> html-playground -> “clone to edit”
  • name “html-playground”, HTML5, create workspace
  • you can now work with the code! fix or add some code
  • double check it in c9 like you would a normal website
  • add it, commit it, (see above) and type 
    git push
    (enter your username and password)
  • go back to https://github.com/sgelbart/html-playground
  • click "pull requests" -> "new pull request"
  • master -> master

Advanced Challenges

Pick and chose from the advanced coding challenges!

  • Play with Chrome Developer tools!
    To open right click on page and go to "Inspect". On the top left corner of the popup there will be a box with an arrow. Click on that to
  • Create a List (you can make it nested if you want!): https://www.w3schools.com/html/html_lists.asp
  • Set a custom color via HEX: https://www.w3schools.com/css/css_colors.asp
  • Set background color and/or image on your page:
    • W3 Schools
    • Site for free images (don't forget attribution)
    • Another site for free images
    • Background pattern creator
  • Add borders to an element (a div, a picture, etc)
  • Use Padding to give some space around your border
  • Use margin to give some space between your border and what's around it
  • Make your background color semi-transparent using RGBA and Dev tools
  • Use Shorthand syntax for margins/padding (if you haven’t already) e.g., margin: 10px 60px;
  • Make a link into a button
